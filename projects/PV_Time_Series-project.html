Page View Time Series Visualizer¶

I did this project as part of Data Analysis with Pyhton certificate from freeCodeCamp. For this project I visualized time series data using a line chart, bar chart, and box plots. Also, I used Pandas, matplotlib, and seaborn to visualize a dataset containing the number of page views each day on the freeCodeCamp.org forum from 2016-05-09 to 2019-12-03. The data visualizations to help understand the patterns in visits and identify yearly and monthly growth.
